
# Practical Programming - Exception Handling and OOP Mastery

## Part 1: Exception Handling
This folder contains the calculator program that handles exceptions like division by zero and invalid input. A custom exception for unsupported operations is also included.

## Part 2: OOP Concepts
This folder contains the Vehicle Management System, demonstrating inheritance, constructors, and method overriding.

## Structure
- `Exception_Handling/CalculatorProgram.java`: The calculator program.
- `OOP_Concepts/VehicleManagementSystem.java`: The Vehicle Management System program.

Instructions on how to run the programs are provided in each folder.
